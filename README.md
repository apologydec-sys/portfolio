# Blessing Akanni — Portfolio (Vite + React + Tailwind)

This is a starter portfolio site scaffolded for Blessing Akanni. It includes animated hero, about, skills, timeline, projects, tips, and contact sections.

Quick setup

```bash
cd "c:/Users/KwEkU kAy/Desktop/portfoloi"
npm install
npm run dev
```

Notes
- Install dependencies then run `npm run dev`.
- Deploy with Vercel or Netlify by connecting the repo.

Deployment suggestions

- Vercel: connect your GitHub repo and push main branch. Vercel auto-detects Vite/React projects.

	1. Create a GitHub repo and push this project.
	2. Connect the repo on Vercel and choose the root project.
	3. Build command: `npm run build`, Output directory: `dist`.

- Netlify: drag & drop `dist` after `npm run build` or connect repo and set the same build settings.

- GitHub Pages (simple): use `gh-pages` package or deploy `dist` branch. For Vite React, recommended: use Vercel or Netlify for best compatibility.
